MAKING LIST

Burda sıralı liste oluşturmaya başlayacağız :

Sıralı liste için -ol- yani -ordered list- tagını kullanıyoruz. bu tag kapanışı olan bi tag tır.

Listelerin herbir maddesi için -li- yani -list- tag ı kullanılır

Making List

    Making ordered list

  1. Sıralı listenin ilk elemanı
  2. Sıralı listenin ikinci elemanı
  3. Sıralı listenin üçüncü elemanı
  4. Sıralı listenin dördüncü elemanı

Sırası önemli olmayan listeler için -ul-yani -unordered list- tg ı kullanılır

    Making unordered list

  • Sırası önemli olmayan listenin ilk elemanı
  • Sırası önemli olmayan listenin ikinci elemanı
  • Sırası önemli olmayan listenin üçüncü elemanı

Burada sayfaya resim ekleme öreniği yaptık

Resim eklemek için -img- tag ı kullanılır ve burda resmin boyutlarını width, height ile değiştirebiliyoruz

Burda sayfaya link ekleme örneği yaptık
