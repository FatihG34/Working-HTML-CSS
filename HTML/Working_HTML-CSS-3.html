Giriş öğesinin içinde bir liste niteliği ve veri listesi öğesinin içinde bir id niteliği kullanırız.
< input> öğesinin, < input> öğesinin list özniteliği ve < datalist> öğesinin id özniteliği aracılığıyla < datalist> öğesiyle ilişkilendirilmesi zorunludur.
